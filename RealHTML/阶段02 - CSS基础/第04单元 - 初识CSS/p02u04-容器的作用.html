课题介绍

本课题取材于实际生活。 在 2020 年，网络已经成为了现在生活及办公不可或缺的一部分，网络的质量可能会直接影响到人们的工作效率。 尤其是在 2020 年初疫情爆发的时候，远程办公几乎成为所有从业者唯一的选择。

一个优秀的公司或企业网络应当有以下特点：

1）设定好公司或企业内部的权限设定，如文件共享等；

2）公司或企业内部的 IP 地址采用自动获取的形式；

3）设置远程网络，方便员工从家中访问公司内部，方便进行配置或获取信息，以达到远程办公的需求。

综上所述，这些特点需要学生们掌握网络规划的基础知识，包含：IP 地址的分配；路由器、交换机的配置，ACL 的配置等。 学生可将课上所学的各式各样的 CCNA 的知识付诸于实践，培养和锻炼学生的自学能力以及解决问题的能力。

在配置过程中，撰写论文，同上掌握资料检索的能力。

本课题成果形式为应用方案。
